# SeaWater-Yaquina



**Objectives**

* compile raw historical data collected in the lower Yaquina River of Newport Oregon, including the modern multi-sensor [Hatfield Coastal Monitoring Station]([Coastal Monitoring Station | Hatfield Marine Science Center | Oregon State University](https://hmsc.oregonstate.edu/research-support/coastal-monitoring-station)) at the OSU Hatfield Marine Science Center
* convert units and QC master files 
* build an open-source UI to visualize parameters and patterns
